# Dart_Xpress
Progressive Web app(PWA) that provide fast & secure courier delivery service to the customers.

## Features 
Push Notification <br>
Courier side earning dashboard<br>
Realtime GPS tracking<br>

## Tech used
Django<br>
Rest API<br>
Google API<br>
Stripe<br>
Paypal<br>
Redis<br>
Channels<br>
